# 以下簡介所需工具

## gdb

調適工具, 可以下斷點查看變數, linux 內建

常用指令

- disas <function name>
- checksec

### gdb-plugin: gdb dashboard

https://github.com/cyrus-and/gdb-dashboard

### gdb-plugin: peda

https://github.com/longld/peda

note: 重點是要用 checksec

## pwntools

python ctf 工具, 可以拿來做為和 nc 互動的工具

https://github.com/Gallopsled/pwntools
